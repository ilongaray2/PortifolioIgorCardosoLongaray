Classificador De Nota Fiscal Eletronica

Script em python que utiliza visão computacional com OpenCV para detectar o tipo de Nota Fiscal Eletrônica.

Scrypt foca uma região do arquivo NFe utilizando visão computacional para pegar o boolean do tipo de nota, Entrada = 0 / Saída = 1.

Com isso ele retorna um json contendo as informações do arquivo lido e se é uma NFe de Venda ou de Compra.
